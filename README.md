## 주제

1. 알록달록 터치 무드등

- 설명: 폼보드로 만든 상자 표면에 여러 개의 터치 센서 를 부착합니다. 아이가 각 센서를 만질 때마다 상자 내부의 RGB LED 또는 네오픽셀 이 해당 영역의 색(빨강, 파랑, 노랑 등)으로 밝게 빛납니다.
- 주요 사용 재료: 폼보드 , TTP223B 아두이노 터치 센서 모듈 , RGB LED 모듈 (또는 WS2812 LED 네오 픽셀 )

2. AI 색깔 맞추기 친구

- 설명: 아이가 물체를 카메라로 보여 줍니다. AI 기능 (노트북): 컴퓨터 비전(CV)이 물체를 인식 합니다.
- 작동: AI가 인식한 카메라정보를 ESP8266에 보내면, 장난감에 내장된 스피커가 해당 물체의 이름을 말해줍니다. LED 도 작동합니다.
- 주요 재료: ESP8266 , 미니 카메라 모듈 , RGB LED 모듈 ,
