## 주제

1. 알록달록 말하는 애착 곰인형

- 설명: 곰인형에 여러 개의 터치 센서 를 부착합니다. 아이가 각 센서를 만질 때마다 상자 내부의 RGB LED 또는 네오픽셀 이 해당 영역의 색(빨강, 파랑, 노랑 등)으로 밝게 빛납니다. 카메라 센서가 정면을 인식해서 정보를 LLM 에게 보낸다. LLM 은 아이가 좋아하는 색깔을 분석해서 지금상황(시간, 조명, 위치)와 쌓여 있는 데이터를 통해 , 곰인형이 아이에게 그 색깔에 대한 이야기를 들려줍니다. 상호작용한 데이터를 데이터베이스로 저장하여 부모가 아이의 선호도를 파악할 수 있도록 합니다.

- 주요 사용 재료: ESP32_cam, 폼보드 , TTP223B 아두이노 터치 센서 모듈 , WS2812 LED 네오 픽셀, 서보 모터, LCD 디스플레이 등
